Font Properties

Jul 14 2019 font a minute de lectura (Alrededor de 183 palabras)

Sets all the font properties in one declaration

font: normal(1) normal(2) normal(3) 40px(4)/1.4em(5) georgia(6), serif(7);

  1. font-style: normal(正體), italic(斜體), oblique(傾寫體);
  2. font-variant: normal, small-caps(小型大寫體);
  3. font-weight: normal, bold, 900;
  4. font-size: 15px, large, 150%;
  5. line-height: normal, 1.2, 1em, 150%,
  6. font-family: “Times New Roman”, Helvetica;
  7. font-family: serif(有襯線，類明體), sans-serif(無襯線，類黑體), cursive(類書寫草體);
    如無支援自設字體，依瀏覽器內建顯示預設值。(再查證)

備註：
italic(斜體)：正確斜體用法，某些字體並無設計。
oblique(傾寫體)：單純把字體傾寫化，不是正規用法，但在需要與正體辨別並沒有斜體時可採用。
